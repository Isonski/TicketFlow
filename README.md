# TicketFlow
TicketFlow is a web application for managing events and tickets. The project follows a three-layer architecture and is developed as part of a course assignment.

Project created by:
- Elitsa Sotirova
- Tihomir Petrov
- Dariya Slavova
# Technologies Used
- C#
- ASP.NET Core MVC
- Entity Framework Core
- ASP.NET Core Identity
- SQL Server
- Razor Views
- Dependency Injection
# Usage
- Create, edit, delete events
- Manage ticket types and availability
-Add categories, descriptions, dates and locations
-View detailed event pages

# Testing
-Unit tests for core business logic
-Controller tests
-InMemory database for realistic scenarios
-Verified CRUD operations
